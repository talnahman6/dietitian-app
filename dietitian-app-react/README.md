# Dietitian App React Native

גרסת React Native / Expo נפרדת לאפליקציית מירי.

הפרויקט הקיים לא נדרס. התיקיה הזו עומדת בפני עצמה.

## בדיקה מהירה

הקובץ `index.html` הוא Preview וובי ל-GitHub Pages.

## הרצה כאפליקציה בעתיד

כאשר תהיה סביבת npm זמינה:

```bash
npm install
npm run web
npm run android
```

## חיבור דאטה

ה־Preview משתמש ב-localStorage וב-Supabase לאותו מאגר מאכלים מותאמים:

`user_data / __global_custom_foods__`
