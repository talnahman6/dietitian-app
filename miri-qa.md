# Miri Chat — שאלות ותשובות

## סטטוס יומי

| שאלה לדוגמה | תשובה |
|---|---|
| כמה קלוריות אכלתי? | אכלת X קל׳ מתוך Y. נשאר לך Z קל׳ + הערת קואוצ׳ינג |
| כמה נשאר לי? | נשאר: X קל׳ / חלבון / פחמימות / שומן + הערת קואוצ׳ינג |
| כמה פחמימות אכלתי היום? | Xg מתוך Yg. נשאר Zg. |
| כמה חלבון אכלתי היום? | Xg מתוך Yg. נשאר Zg + הצעת מזון אם נשאר ≤30g |
| כמה שומן אכלתי היום? | Xg מתוך Yg. נשאר Zg. |
| מה המצב? / עדכן אותי / איפה אני עומד? | סיכום מלא % + הערת קואוצ׳ינג |
| כמה אחוז מהיעד? | X% קלוריות, Y% חלבון |
| מה היעד שלי? | X קל׳ / Xg חלבון / Xg פחמימות / Xg שומן |
| מה המטרה שלי? | ירידה / עלייה + TDEE |

## לוגיקת קואוצ׳ינג (חוקיות)

| מצב | הערת מירי |
|---|---|
| נשאר ≤80 קל׳ | "רק עוד X קל׳ ואז עמדת ביעד! כמעט שם 🎯" |
| הגיע בדיוק ליעד | "עמדת ביעד היום! מדהים אתה! 🌟🎉" |
| עבר ב-1-300 קל׳ | "X קל׳ מעל היעד — קורה לכולם, מחר יותר מדויק 🎯" |
| עבר ב->300 קל׳ | "גם יום כזה קורה 😊 מחר מתחילים מחדש!" |
| 70-80% ביעד הקלוריות | "X% ביעד — ממשיכים חזק! 💪" |
| נשאר ≤30g חלבון | "נשאר Xg חלבון — [הצעת מזון ספציפית]" |
| חלבון הושלם, קל׳ נשארו | "חלבון ✅ השגת! עכשיו להשלים קלוריות 🔥" |
| חלבון <30% ו>50% קל׳ | "שים לב — עדיין חסר הרבה חלבון! 💪" |

## הצעות מזון לפי חלבון שנשאר

| נשאר חלבון | הצעה |
|---|---|
| ≤6g | ביצה אחת (~6g) 🥚 |
| ≤10g | יוגורט יווני 100g (~10g) 🥛 |
| ≤12g | קוטג׳ 100g (~12g) |
| ≤17g | יוגורט יווני 170g (~17g) 🥛 |
| ≤25g | קופסת טונה (~25g) 🐟 |
| ≤31g | 100g חזה עוף (~31g) 🍗 |
| >31g | מנת חלבון גדולה — עוף / טונה / ביצים |
